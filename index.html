Privacy Policy

VoiceQA is an offline-first Android application. Your privacy is important to us.

Data Collection

VoiceQA does not collect, transmit, or store any personal data on external servers. All processing occurs entirely on your device.

  • No analytics or telemetry are collected
  • No network requests are made by the app
  • No cloud services are used
  • No user accounts are required

Voice Processing

Voice recognition is performed entirely on-device using the Vosk offline speech recognition engine. No audio recordings or transcripts are sent to any server.

Local Data Storage

The app stores the following data locally on your device:

DataLocationPurpose
Settings (reviewer name, preferences)SharedPreferencesPersist user configuration
Verdict logs (JSON/CSV)Project folder on device storageRecord QA review decisions
Session metadataProject folder on device storageEnable session resume

You can delete all stored data by clearing the app's data in Android Settings or by deleting the verdict/session files from your project folders.

Permissions

PermissionWhy It's Needed
RECORD_AUDIOMicrophone access for offline voice command recognition via Vosk
MANAGE_EXTERNAL_STORAGERead audio files for playback and move them into verdict folders (approve/reject/review)
READ_EXTERNAL_STORAGERead audio files for playback
WRITE_EXTERNAL_STORAGEWrite verdict log files (JSON/CSV) to the project folder
READ_MEDIA_AUDIOAndroid 13+ scoped storage access for audio files
WAKE_LOCKPrevent the screen from turning off during hands-free review sessions

No permission is used for any purpose other than what is listed above.

Third-Party Libraries

The app uses open-source libraries. None of these libraries collect or transmit user data in the way they are configured in this app.

Changes to This Policy

Any changes to this privacy policy will be documented in the project's CHANGELOG.

Contact
